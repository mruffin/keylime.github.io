Want to try Keylime?

Get the Ansible role with a TPM emulator


$ git clone https://github.com/keylime/keylime-vagrant-ansible-tpm-emulator

Use our Vagrantfile to stand up a test VM


$ vagrant up --provision
$ vagrant ssh
$ sudo -s

Start the verifier


# keylime_verifier
Using config file /etc/keylime.conf

Start the registrar
